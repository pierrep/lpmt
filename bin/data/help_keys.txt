KEY COMMANDS:
=============

'h' - display this help dialog

'a' - adds new quad
'CTRL-x' - delete selected quad
'>' - go to next quad
'<' - go to previous quad
'+' - raise active quad position in layers pile
'-' - lower active quad position in layers pile
'q' - fills window with active quad
'1' - shows general settings page of gui

'g' - toggles gui (for quad warping with mouse gui must be switched off)
'f' - toggles fullscreen mode
'w' - toggles cam snapshot window background
'm' - toggles mask-editing mode
'b' - toggles surface bezier/grid warping editing mode

'spacebar' - toggles projection/setup modes (in Timeline interface, space starts/stops the timeline playback
'r' - resyncs all videos, color transitions and slideshows in all quads to starting point
'p' - starts projection
'o' - stops projection

'F1' - selects general settings gui page
'F2' - selects first gui page for active quad settings
'F3' - selects second gui page for active quad settings
'F4' - selects third gui page for active quad settings
'F9'     - toggles timeline BPM reference grid
'F10'    - toggles timeline interface
'F11'    - toggles stage visibility under timeline
'F12'    - play/stop timeline
'CTRL-q' - Quits the application
